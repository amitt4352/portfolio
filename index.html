About

A designer who codes, and a developer who designs.

I'm a Lead UI/UX Designer & Frontend Developer based in Indore, India, with 12+ years of experience turning complex requirements into pixel-perfect, responsive, and accessible web interfaces. I work end to end — from wireframes and interactive prototypes in Figma to production-ready, responsive code in HTML, CSS/SCSS, JavaScript and jQuery.

Over my career I've delivered 500+ screens and reusable design systems across enterprise products. More recently, I ship production React applications using AI-assisted development workflows (Cursor, GitHub Copilot, Claude) and direct and review my team's React work to keep frontends complete, polished, and consistent.

I lead a team of 2 UI/UX designers and 3 React developers within Agile/Scrum, run design and code reviews, and care deeply about accessibility, performance, and consistency across every screen.
